## 文件详情

### 1. response_prompts.py ✨ 新增

**用途**: ResponseAgent 生成最终用户响应

**提示词列表**:
- `CHAT_SYSTEM_PROMPT` - 普通对话系统提示词
- `GREETING_RESPONSE` - 问候响应模板
- `GOODBYE_RESPONSE` - 告别响应模板
- `CLARIFICATION_TEMPLATE` - 澄清请求模板
- `ERROR_RESPONSE` - 错误响应模板
- `NO_IMAGE_RESPONSE` - 无图片响应
- `DETECTION_RESULT_TEMPLATE` - 检测结果格式化
- `DISEASE_FEATURE_TEMPLATE` - 病害特征模板
- `TREATMENT_TEMPLATE` - 防治建议模板
- `KNOWLEDGE_RESPONSE_TEMPLATE` - 知识响应模板

**使用场景**:
- 普通对话生成
- 问候和告别
- 检测结果呈现
- 知识查询响应

### 2. context_prompts.py ✨ 新增

**用途**: ContextAgent 管理对话上下文

**提示词列表**:
- `CONTEXT_SYSTEM_PROMPT` - 上下文管理系统提示词
- `CONTEXT_EXTRACTION_TEMPLATE` - 上下文提取模板

**使用场景**:
- 多轮对话理解
- 指代关系识别
- 省略信息补全
- 话题转换追踪

### 3. memory_prompts.py ✨ 新增

**用途**: MemoryAgent 管理对话记忆

**提示词列表**:
- `MEMORY_SYSTEM_PROMPT` - 记忆管理系统提示词
- `MEMORY_SUMMARY_TEMPLATE` - 记忆摘要模板
- `MEMORY_COMPRESSION_TEMPLATE` - 记忆压缩模板

**使用场景**:
- 对话历史存储
- 记忆检索
- 历史压缩
- 信息清理

### 4. validation_prompts.py ✨ 新增

**用途**: InputValidationAgent 和 ResultValidationAgent

**提示词列表**:
- `INPUT_VALIDATION_SYSTEM_PROMPT` - 输入验证系统提示词
- `INPUT_VALIDATION_TEMPLATE` - 输入验证模板
- `RESULT_VALIDATION_SYSTEM_PROMPT` - 结果验证系统提示词
- `RESULT_VALIDATION_TEMPLATE` - 结果验证模板
- `CLARIFICATION_TEMPLATES` - 澄清请求模板集
  - `no_image` - 无图片澄清
  - `vague_question` - 模糊问题澄清
  - `missing_context` - 缺少上下文澄清
  - `invalid_format` - 格式错误澄清

**使用场景**:
- 输入完整性检查
- 结果质量验证
- 澄清请求生成
- 质量评分

## 使用方式

### 方式 1: 直接导入

```python
from yoloapp.prompt import CHAT_SYSTEM_PROMPT

messages = [
    Message.system_message(CHAT_SYSTEM_PROMPT),
    user_message
]
```

### 方式 2: 批量导入

```python
from yoloapp.prompt import (
    CHAT_SYSTEM_PROMPT,
    GREETING_RESPONSE,
    GOODBYE_RESPONSE,
)
```

### 方式 3: 模块导入

```python
from yoloapp.prompt import response_prompts

response = response_prompts.GREETING_RESPONSE
```

## 维护指南

### 添加新提示词

1. 确定提示词用途和所属文件
2. 按照模板编写提示词
3. 在 `__init__.py` 中导出
4. 更新本文档

### 优化现有提示词

1. 收集用户反馈
2. 分析响应质量
3. 调整提示词内容
4. A/B 测试验证
5. 更新文档

### 版本管理

- 使用 Git 追踪变更
- 重大修改创建新版本
- 保留历史版本备份
- 记录变更原因
