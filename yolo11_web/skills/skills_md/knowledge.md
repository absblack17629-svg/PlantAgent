# Skill: 农业知识问答 (Agricultural Knowledge Q&A)

## 描述
基于本地知识库和 RAG (检索增强生成) 技术，回答关于水稻种植、病虫害防治、农业技术等问题。

## 参数
- `question` (必填): 用户的问题文本
- `detections` (可选): 如果之前进行了病害检测，传入检测结果以获得更精准的防治建议
- `top_k` (可选): 返回最相关的知识条目数量，默认为 3

## 返回值格式
返回 JSON 格式的回答结果，包含:
- `success`: 是否成功获取答案 (boolean)
- `answer`: 直接回答用户问题的文本 (string)
- `sources`: 参考的知识来源数组 (array)
- `context_used`: 用于生成回答的上下文片段 (array)

## 使用示例
```
用户问: "稻瘟病怎么防治？"
1. 调用 skill 时传入 question="稻瘟病怎么防治？"
2. 返回结果: {"success": true, "answer": "稻瘟病的防治方法包括...","sources": ["水稻病害防治手册.pdf"], "context_used": ["选用抗病品种...", "合理施肥..."]}
```

## 注意事项
- 知识库内容主要来源于水稻病害手册、农业技术指南等专业文档
- 如果知识库中没有相关信息，系统会返回"无法找到相关信息"
- 传入之前的检测结果可以获得针对性的防治建议
- 回答会优先使用最新的研究成果和实践经验
