# ✨ 提示词工程优化

## 优化时间
2026-03-08

## 优化目标

参考 OpenManus 的提示词工程实践，为 yoloapp 的每个使用 LLM 的 Agent 创建专业、独立的提示词文件。

## OpenManus 提示词工程特点

### 1. 模块化设计
- 每个 Agent 有独立的提示词文件
- 提示词与代码分离
- 便于维护和优化

### 2. 结构化提示词
- 清晰的角色定义
- 明确的职责说明
- 具体的输出要求

### 3. 专业性
- 针对特定任务优化
- 包含详细的指导原则
- 提供示例和模板

## 新增提示词文件

### 1. response_prompts.py - 响应生成提示词

**用途**: ResponseAgent 生成最终用户响应

**核心提示词**:
- `CHAT_SYSTEM_PROMPT`: 普通对话系统提示词
- `GREETING_RESPONSE`: 问候响应模板
- `GOODBYE_RESPONSE`: 告别响应模板
- `NO_IMAGE_RESPONSE`: 无图片响应
- `DETECTION_RESULT_TEMPLATE`: 检测结果格式化模板
- `TREATMENT_TEMPLATE`: 防治建议模板

**特点**:
```python
CHAT_SYSTEM_PROMPT = """你是一个友好、专业的水稻病害智能助手。

## 你的身份
- 名称：水稻病害智能助手
- 专长：水稻病害诊断、防治建议、农业知识咨询
- 特点：专业、友好、实用

## 你的能力
1. **病害诊断**：分析图片，识别水稻病害
2. **防治建议**：提供科学的防治方案
3. **知识咨询**：回答农业相关问题
4. **友好对话**：进行自然的对话交流

## 对话原则
1. **专业优先**：对于农业相关问题，提供专业、准确的回答
2. **友好态度**：保持友好、耐心的交流态度
3. **实用导向**：提供可操作的建议和方案
4. **适度引导**：对于非农业问题，友好提醒你的专长领域
...
"""
```

### 2. context_prompts.py - 上下文管理提示词

**用途**: ContextAgent 管理对话上下文

**核心提示词**:
- `CONTEXT_SYSTEM_PROMPT`: 上下文管理系统提示词
- `CONTEXT_EXTRACTION_TEMPLATE`: 上下文提取模板

**特点**:
- 明确上下文管理职责
- 提供提取策略
- 定义输出格式

### 3. memory_prompts.py - 记忆管理提示词

**用途**: MemoryAgent 管理对话记忆

**核心提示词**:
- `MEMORY_SYSTEM_PROMPT`: 记忆管理系统提示词
- `MEMORY_SUMMARY_TEMPLATE`: 记忆摘要模板
- `MEMORY_COMPRESSION_TEMPLATE`: 记忆压缩模板

**特点**:
- 定义记忆策略
- 提供压缩方法
- 优化存储效率

### 4. validation_prompts.py - 验证提示词

**用途**: InputValidationAgent 和 ResultValidationAgent

**核心提示词**:
- `INPUT_VALIDATION_SYSTEM_PROMPT`: 输入验证系统提示词
- `RESULT_VALIDATION_SYSTEM_PROMPT`: 结果验证系统提示词
- `CLARIFICATION_TEMPLATES`: 澄清请求模板集

**特点**:
- 明确验证规则
- 提供质量评分标准
- 友好的澄清请求

### 5. rag_prompts.py - RAG 提示词（优化）

**优化内容**:
- 添加 `RAG_SYSTEM_PROMPT`: 专业的农业知识专家提示词
- 优化 `RAG_QA_TEMPLATE`: 更详细的回答要求
- 优化 `DETECTION_ANALYSIS_TEMPLATE`: 结构化的分析模板
- 新增 `KNOWLEDGE_RETRIEVAL_PROMPT`: 知识检索策略
- 新增 `NO_RELEVANT_INFO_RESPONSE`: 无相关信息响应
- 新增 `PARTIAL_INFO_RESPONSE_TEMPLATE`: 部分信息响应

**优化前**:

## 使用示例

### 示例 1: ResponseAgent 生成普通对话

```python
from yoloapp.prompt import CHAT_SYSTEM_PROMPT

async def _generate_chat_response(self) -> str:
    llm = get_llm_client()
    
    messages = [
        Message.system_message(CHAT_SYSTEM_PROMPT),
        user_message
    ]
    
    response = await llm.ask(messages)
    return response
```

### 示例 2: RAGAgent 生成知识回答

```python
from yoloapp.prompt import RAG_SYSTEM_PROMPT, RAG_QA_TEMPLATE

async def query_knowledge(self, question: str) -> str:
    # 检索知识
    docs = await self.retrieve(question)
    context = "\n\n".join(docs)
    
    # 构建提示
    prompt = RAG_QA_TEMPLATE.format(
        context=context,
        question=question
    )
    
    # 调用 LLM
    messages = [
        Message.system_message(RAG_SYSTEM_PROMPT),
        Message.user_message(prompt)
    ]
    
    response = await llm.ask(messages)
    return response
```

### 示例 3: 使用响应模板
